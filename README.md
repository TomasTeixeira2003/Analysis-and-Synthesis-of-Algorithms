## Project 1 - Rectangular Area Tiling

A combinatorial optimization problem where the goal is to calculate the total number of unique ways to tile a "staircase" grid area using only square tiles of various dimensions (e.g., $1 \times 1, 2 \times 2, 3 \times 3$). It focuses on Backtracking and Memoization to handle large state spaces efficiently.

## Project 2 - High-Speed Railway Optimization

A network design problem aimed at maximizing commercial exchange between regions while minimizing infrastructure costs. It involves identifying potential connections through Delaunay Triangulation and selecting the optimal set of tracks using a Minimum Spanning Tree (MST) algorithm.
